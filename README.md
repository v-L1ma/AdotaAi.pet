<h2 id="sobre">📌 Sobre</h2>

O Adota ai é um site criado com a finalidade de facilitar o processo de adoção e de doação de animais, um site onde existem funcionalidades tanto para o adotante quanto para o doador.

<a href="https://adotaai-pet.netlify.app/" target="_blank">[Link para o projeto]</a> 

<h2 id="funcionalidades">🚀 Funcionalidades</h2>

Aqui você irá encontrar as principais funcionalidades do sistema.

<ul>
 <li>Cadastrar um animal para adoção.</li>
 <li>Exibir todas os animais disponiveis para adoção.</li>
 <li>Ver o perfil completo de um animal.</li>
 <li>Pagina com ONGS</li>
 <li>Editar informações de um animal cadastrado.</li>
 <li>Excluir o cadastro de um animal.</li>
</ul>

<h2 id="rotas">📍 Rotas da aplicação</h2>

Lista de todas as rotas existente no site:
​
| Rota                                        | Descrição                                                 |
|--------------------------------------------|------------------------------------------------------------|
| <kbd>/</kbd>                               | Página inicial                                             |
| <kbd>/adotar</kbd>                         | Página de login                                            |
| <kbd>/adotar/animal/:id</kbd>              | Perfil completo de um animal.                              |
| <kbd>/ongs</kbd>                           | Página de cadastro                                         |
| <kbd>/doar</kbd>                           | Portal de vagas                                            |
| <kbd>/cadastro</kbd>                       | Pagina para cadastrar um animal                            |
| <kbd>/editar</kbd>                         | Pagina para editar informações de um animal                |
